# Análise Estatística de Produtos da Amazon

Projeto acadêmico desenvolvido em Python para análise estatística e visualização de dados.

## Objetivo
Analisar preços, avaliações e comportamento dos consumidores a partir de um conjunto de dados da Amazon.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Git / GitHub

## Conteúdos abordados
- Limpeza e pré-processamento de dados
- Análise estatística descritiva
- Visualização de dados
